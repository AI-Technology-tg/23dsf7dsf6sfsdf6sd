# Как залить на Netlify (папка уже готова)

## Важно про ключи

При **перетаскивании папки** Netlify **не подставляет** переменные из `.env` в функции автоматически. Нужно **один раз** добавить их в панели сайта (или через CLI ниже).

Файл **`.env`** в этой папке уже заполнен — откройте его и скопируйте три строки в Netlify.

## Шаги

1. Откройте <https://app.netlify.com/drop> (или Sites → Add site → Deploy manually).
2. Перетащите **всю папку `netlify-minko`** (эту папку, не весь BOT).
3. Дождитесь зелёного деплоя. Скопируйте URL вида `https://xxxx.netlify.app`.
4. **Site configuration** → **Environment variables** → **Add a variable** (или **Import from .env**):
   - `MINKO_FREE_API_KEY`
   - `OPENAI_API_KEY`
   - `XAI_API_KEY`  
   Значения — из файла `.env` здесь же.
5. **Deploys** → **Trigger deploy** → **Clear cache and deploy site** (чтобы функция подхватила ключи).

## Сайт Re-Minko

В **`config.local.js`** основного сайта укажите (подставьте свой хост с шага 3):

```js
window.APP_CONFIG.minkoChatProxy = 'https://ВАШ-ПОДДОМЕН.netlify.app/chat';
```

## Альтернатива: CLI (подтянет `.env` сам)

```bash
cd netlify-minko
npx netlify-cli login
npx netlify-cli init
npx netlify-cli env:import .env
npx netlify-cli deploy --prod
```

После `init` выберите созданный сайт или создайте новый.
