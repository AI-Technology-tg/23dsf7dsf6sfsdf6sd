Minko AI — облачный прокси

Этот сайт на Netlify отдаёт POST /chat для страницы Minko AI на Re-Minko.

Проверка «в сети» в чате использует GET / (эта страница — ответ 200).

В config.local.js основного сайта задайте:

window.APP_CONFIG.minkoChatProxy = 'https://<ваш-поддомен>.netlify.app/chat';

Переменные окружения задаются в Netlify → Site settings → Environment variables (см. README в этой папке).
